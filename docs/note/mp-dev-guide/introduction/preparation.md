微信小程序 开发准备（申请 AppID、微信开发者工具、新建项目）。

## 申请 AppID

* [微信公众平台](https://mp.weixin.qq.com/)  
  注册，账号类型选择小程序  
  登录
* 开发 -> 开发设置 AppID

## [微信开发者工具](https://mp.weixin.qq.com/debug/wxadoc/dev/devtools/download.html)

## 新建项目

* 微信开发者工具
* 小程序项目 -> 小程序
* 新建
  * 项目名称
  * 目录  
    空目录
  * AppID 使用测试号  
    或输入注册的 AppID，后端服务选择不使用云服务
* 可以扫描预览二维码预览
