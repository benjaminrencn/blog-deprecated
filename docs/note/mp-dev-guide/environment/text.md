| 属性                  | 类型     | 说明                                                           |
| -------------------- | -------- | ------------------------------------------------------------- |
| `identifier`         | `Number` | 触摸点的标识符                                                  |
| `pageX`, `pageY`     | `Number` | 距离文档左上角的距离，文档的左上角为原点 ，横向为 X 轴，纵向为 Y 轴     |
| `clientX`, `clientY` | `Number` | 距离页面可显示区域（屏幕除去导航条）左上角距离，横向为 X 轴，纵向为 Y 轴 |
