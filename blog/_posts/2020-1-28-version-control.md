# 版本管理概述

* 版本控制系统 （VCS，version control system）
  * 记录文件修订记录
  * 查阅、回到某个历史版本
* 分类
  * “人肉” VCS
  * LVCS 本地
    * RCS （Revision Control System）
  * CVCS 集中式
    * CVS (Concurrent Versions System)
    * SVN (Subversion)
    * Perforce
  * DVCS 分布式
    * Git
    * Mercurial
