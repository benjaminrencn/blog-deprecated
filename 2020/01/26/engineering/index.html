# 示例

# Html5 Boilerplate

.
├─ css  // 样式
│  ├─ main.css      // 页面样式
│  └─ normalize.css // reset css
├─ img        // 图片
├─ index.html // 入口文件
└─ js
   ├─ main.js     // 页面逻辑代码
   ├─ plugins.js  // 插件代码 工具 
   └─ vendor      // 库文件
      ├─ jquery-1.10.2.min.js
      └─ modernizr-2.6.2.min.js

# Web Starter Kit

.
└─ app
   ├─ images  // 图片
   │  └─ hamburger.svg 
   ├─ index.html  // 入口文件
   ├─ script
   │  └─ main.js  // 页面逻辑代码
   └─ styles
      └─ main.css // 页面样式
