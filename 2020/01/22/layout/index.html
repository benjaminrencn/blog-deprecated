# table

    <p>left</p>
  </div>
  <div class="right">
    <p>right</p>
    <p>right</p>

  table-layout: fixed;
  width: 100%;
}

}

box-sizing

# table

  <div class="left">
    <p>left</p>
  </div>
  <div class="right">
    <p>right</p>
    <p>right</p>
  </div>

  width: 100%;
}

  display: table-cell;

  border-right: 20px solid transparent;

left

right

right

提示

background-clip

值 说明
border-box 默认值 背景延伸到 border
padding-box 背景延伸到 padding
content-box 背景裁剪到内容区（content box）
text 背景裁剪成文字的 color
