Vue.js 模板语法 插值。

# 模板语法

  • Vue 使用了基于 HTML 的模板语法
    允许声明式地将 DOM 绑定至底层 Vue 实例的数据

# 插值

# 文本

  • Mustache 语法（双大括号）

    <span>Msg: {{ msg }}</span>
    1

  • 指令 v-once
    只渲染元素、组件一次

    <span v-once>Msg: {{ msg }}</span>
    1

# HTML

  • 指令 v-html
    更新 innerHTML

    <div v-html="html"></div>
    1

# 属性

  • 指令 v-bind
    动态绑定属性、组件 prop。

    <img v-bind:src="imageSrc">
<!-- 缩写 -->
<img :src="imageSrc">
    1
    2
    3

# JavaScript 表达式

  • 单个表达式

    {{ number + 1 }}
{{ ok ? 'YES' : 'NO' }}
{{ message.split('').reverse().join('') }}
