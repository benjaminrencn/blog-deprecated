# 背景

  1. 多端泛滥
  2. 体验不好
  3. 生态贫瘠

# 产品特征

  1. 跨品台更多
    保留不同平台特殊功能
  2. 运行体验更好
    组件、ap与微信小程序一致 兼容 weex 原生渲染
  3. 通用技术栈
    vue的语法、微信小程序的 api 内嵌 mpvue
  4. 开放的生态
    1. 支持 npm
    2. 支持微信小程序自定义组件及 SDK
    3. 兼容 mpvue
    4. app 端支持和原生混合编码
    5. DCloud 插件市场
