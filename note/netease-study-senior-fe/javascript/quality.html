# 单元测试

# 概念

  • 测试是验证代码时候可以按预期工作的方法
  • 单元测试是对最小可测试单元进行验证

# 意义

  • 检测出潜在 bug
  • 快速反馈功能输出、验证代码是否能达到预期
  • 保证代码重构安全性
  • 方便协作开发
