  • YAGNI 原则（You Ain’t Gonna Need It）
    适可而止。不做过度设计。只做必需功能，不做你认为可能需要的功能。

  • DRY 原则（Don't Repeat Yourself）
    不重复自身

<template>
  <div>
    <div v-if="loading">加载中...</div>
    <slot v-else :data="data"></slot>
  </div>
</template>

<script>
export default {
  props: ['url'],
  data() {
    return {
      loading: true,
      data: {},
    }
  },
  created() {
    setTimeout(() => {
      this.loading = false
      this.data = { name: 'Li Lei' }
    }, 1000)
  },
}
</script>
