# CSS Reset

#网易云课堂# #高级前端开发工程师# #前端基础预习课# #页面架构# #CSS Reset# #CSS#

背景，目的，延伸历史演变、解决方案。

# 背景

  1. 浏览器有默认样式（user agent stylesheet），且不同浏览器默认样式不同
  2. 不符合产品设计

# 作用

  1. 清除默认样式
  2. 定义全局样式
