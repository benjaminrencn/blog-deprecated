# Html5 Boilerplate

.
├─ css  // 样式
│  ├─ main.css      // 页面样式
│  └─ normalize.css // reset css
├─ img        // 图片
├─ index.html // 入口文件
└─ js
   ├─ main.js     // 页面逻辑代码
   ├─ plugins.js  // 插件代码 工具 
   └─ vendor      // 库文件
